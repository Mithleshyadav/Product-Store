# Product Store

A simple product store project built with React and Vite.

---

## Steps to Run

1. Clone the repository:

```bash
https://github.com/Mithleshyadav/Product-Store.git
```

2. Navigate to the project folder:

```bash
cd Product-Store
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

5. Open the app in your browser (usually at `http://localhost:5173`).

---

* Product API: VITE_API_URL = 'https://fakestoreapi.com'
